PRESETS:

Enterprise AI orchestration ships as four composable primitives that work together to run complex automations over retrieved content. The same pattern powers internal tools at major studios for metadata extraction, localization packaging, and workflow automation.

◆ PRIMITIVE 01

Agent Router

Registers named agent functions and dispatches calls by name. Each agent is a Python callable that accepts a payload dict and returns a result — swap LLM calls for deterministic logic without changing workflow YAML.

◆ PRIMITIVE 02

Tool Registry

Named utility functions callable via kwargs. Clean-text, word-count, API wrappers, database readers — the deterministic building blocks that agents string together in multi-step pipelines.

◆ PRIMITIVE 03

RAG Layer

TF-IDF document indexer with cosine-similarity retrieval. Returns ranked documents plus an auto-summary built from leading sentences. Drop-in ready for swapping to dense embeddings (OpenAI, Cohere, sentence-transformers).

◆ PRIMITIVE 04

Workflow Engine

YAML-declared step graph that resolves {{variable}} references between steps. Each step invokes either an agent or a tool. Full execution trace is captured in history for debugging and audit.

◆ API: POST /rag/query with {query, top_k}  ·  POST /workflow/run with {workflow} (YAML string)  ·  POST /rag/index with {documents}  ·  GET /history
